# real-world-vue

## Project setup

```
$ yarn install
```

### Running a mock REST API server

```
$ yarn global add json-server
 $ json-server --watch db.json
```

### Compiles and hot-reloads for development

```
$ yarn run serve
```

### Compiles and minifies for production

```
$ yarn run build
```

### Run your tests

```
$ yarn run test
```

### Lints and fixes files

```
$ yarn run lint
```
